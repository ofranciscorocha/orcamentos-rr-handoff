# orcamentos-rr-handoff

Pacote de handoff para emitir e organizar os orcamentos do Cilia (Bradesco) num desktop Windows, seguindo as regras da Recicladora Rocha.

Comece pelo GUIA-COMPLETO-WINDOWS.md. As regras de organizacao por estado e operador estao em METODO-ORGANIZACAO-ORCAMENTOS.md. Detalhes tecnicos do Cilia em 01-GUIA-CILIA.md. Os scripts estao na pasta scripts/.
